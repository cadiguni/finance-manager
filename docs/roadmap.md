# FinTrack Roadmap

## Phase 1

- ASP.NET Core Web API backend
- EF Core with PostgreSQL
- CRUD for categories
- CRUD for accounts
- CRUD for transactions
- Monthly dashboard summary endpoint
- Basic React frontend
- Docker Compose
- README with local setup
- Unit tests for application services
- Initial GitHub Actions CI

## Phase 2

- Installment purchase generation
- Recurring transaction generation
- Forecast for upcoming months
- CSV import with preview and validation
- Import history

## Phase 3

- Excel import - in progress
- PDF import
- Credit card statement parsing
- Automatic categorization by keywords - in progress
- Monthly budget by category
- Report export
- JWT authentication
- Cloud deployment
- CI/CD pipeline
- Basic observability
